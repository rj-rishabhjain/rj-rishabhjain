## 🚀 &nbsp;Featured Projects

---

**⚖️ FairLens — Automated Algorithmic Bias Mitigation Platform** &nbsp;&nbsp;`Java` `Data Pipelines` `Statistical Fairness`
> Core backend developer — 4-member team, Google Hackathon 2026. Built the validation pipeline that **pre-screens datasets for statistical bias** before they reach decision-making models. Detects class imbalance, feature distribution skew, and missing-value patterns — and surfaces *which* features are causing unfair outputs so developers can act on it. Modular architecture: every check runs independently with zero side effects.

---

**🌍 National Disaster Relief Centre** &nbsp;&nbsp;`Java EE` `Servlets` `JSP` `PostgreSQL` `Supabase` `PWA` `Service Workers`
> Solo full-stack project. An **offline-first PWA** for disaster relief coordination — field workers log requests with zero connectivity; Service Workers queue them in LocalStorage and sync automatically once online. A Java triage engine auto-ranks requests by severity and vulnerability. GPS geo-fencing blocks invalid submissions. Multi-tenant admin dashboards with live transaction logs. Battery-level detection shifts the UI to minimal SOS mode at low charge.

---

**💌 Digital Khat (डिजिटल ख़त) — Real-Time Messaging Platform** &nbsp;&nbsp;`Java 17` `Spring Boot` `WebSockets` `PostgreSQL` `Redis` `Docker` `Firebase Auth`
> Microservices messaging platform with **WebSocket real-time delivery** and Redis-managed per-user connection state. The signature feature: a **"Midnight Compiler"** — a cron-scheduled engine that consolidates a day's messages into a single permanent digital letter (a time-capsule, not a chat log). JWT + Firebase OTP auth. Encrypted PostgreSQL archival. Fully containerised with Docker.

---

**🔊 DualStream — Desktop Audio Routing Utility** &nbsp;&nbsp;`Python` `Flask` `WASAPI Loopback` `PyInstaller` `JavaScript`
> Windows blocks simultaneous Bluetooth audio to multiple devices. This utility **hooks directly into the WASAPI loopback interface** to capture and duplicate system audio across two endpoints in real time. Flask localhost control panel with live endpoint switching. Packaged via PyInstaller into a single portable `.exe` — zero Python install needed on the target machine.

---

**🤝 Impact Circle — Resource Coordination System** &nbsp;&nbsp;`HTML5` `Java Servlets` `URL Rewriting` `Session Architecture`
> Backend developer — built a closed-loop platform to **modernize volunteer logistics and mutual aid coordination**. Algorithmic resource-management framework with full audit trail visibility. Stateful session management via URL Rewriting across distributed servlets — deliberately chosen over cookies for reliability in variable-connectivity environments.

---

**🌱 Google Solution Challenge 2026** &nbsp;&nbsp;`Team Lead` `UN SDGs` `Cross-functional Team` `Ongoing`
> Leading a four-person team to build a **technology-driven solution aligned with UN Sustainable Development Goals**. Full ownership of technical architecture, sprint planning, code review, and milestone delivery. Coordinating frontend, backend, and deployment streams across the team.
